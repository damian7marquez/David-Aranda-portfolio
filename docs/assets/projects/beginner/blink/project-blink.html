> Blink Without Delay

This project demonstrates basic GPIO control using an Arduino. An LED blinks at a fixed interval using blocking delays.

  • Level: Beginner
  • Microcontroller: Arduino Uno
  • Concepts: GPIO, Timing, Digital Output

Source Code


#define LED_PIN 13

void setup(void) {
    pinMode(LED_PIN, OUTPUT);
}

void loop(void) {
    digitalWrite(LED_PIN, HIGH);
    delay(1000);
    digitalWrite(LED_PIN, LOW);
    delay(1000);
}
